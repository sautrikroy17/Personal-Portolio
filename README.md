# Sautrik Roy — Personal Portfolio

An ultra-premium, highly interactive full-stack developer portfolio. Built with modern web technologies, it features immersive parallax animations, deep custom aesthetics, and incredibly crisp performance.

## 🚀 Live Demo
Visit the live site here: [https://sautrikroy.me](https://sautrikroy.me/)

## ✨ Key Features
- **Fluid UI/UX:** Heavily integrates `framer-motion` for complex staggered scale-ins, native-scroll parallax effects (using `useScroll` & `useTransform`), and immersive page entry animations.
- **Modern Apple-Like Aesthetics:** A deep Navy Blue and Cyan neon style profile crafted entirely with custom CSS gradients and Tailwind CSS v4. Features extensive deep glass-card backdrops and glowing accents.
- **Projects Showcase:** Highlights algorithmic architectures like C++ robust systems (Quizzify AI), high-speed terminal tools (Credit Card Validator), and Hackathon MVP systems (Legacy Lens).
- **Fully Responsive Structure:** Mobile-first fluid CSS Grid layouts ensuring beautiful scaling down to mobile devices. 

## 🛠️ Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React & Custom inline SVGs

## 💻 Local Setup
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/sautrikroy17/Portolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd Portolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser to `http://localhost:5173/` to see the site!
